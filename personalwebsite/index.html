About Me

Collin Chuang

Updated: May 21, 2023

As the only computer scientist in my family, I mostly observe my peers and students my age as guidance for anything technology related. While there are benefits to having family members that are in computer science, having the freedom to explore the vast career paths in computer science on your own can be rewarding, in my opinion. When I wrote my first "Hello World" program in the fall of 2021, it was exciting to have taken my first steps, but also frightening. The first thought that crossed my mind was, "How can this 'Hello World' program ever be of use in the real world?"

So far in my studies in college, I have a solid foundation in programming languages like Python and C++. However, I learned that I can't make any web applications using just Python or C++, so I went ahead and learned the basics of web development. I was able to pick up HTML pretty quickly, but I found that formatting in CSS took too much time. I made the decision to use Bootstrap to be more efficient in styling and make my site look a little bit neater. I then learned that JavaScript can make your web pages more interactive, utilizing the DOM. I also have exposure to lower level concepts such as dynamic memory allocation, pointers, and even programming in MIPS assembly language.

I strive to be a well-rounded individual, believing in the importance of collaboration, communication, and persistence. I'm a quick learner and I'm constantly searching for opportunities to expand my knowledge outside of the classroom. Feel free to reach out/connect with me through any of my media links!

Round Robin Scheduling

Developed scheduling algorithm

Round-robin scheduling is an algorithm used in computer operating systems and is used widely in time-sharing multitasking environments. The scheduler allocates CPU time to each process in a cyclic manner, allowing each process to run for its assigned time and then moving on to the next process in the queue. The algorithm ensures that no single process dominates the CPU for an extended period, ensuring reasonable time allocated for all processes. My implementation of this algorithm consists of using a queue and is written in C++.
